Zoom Color Picker | UX guidelines

Summary

The color picker lets users choose a color from a predefined color palette. See in the following link the way that behaves

Color Picker

What

With the color picker the user can select a color from a range of colors by showing the colors in a popover and showing the selected color in a color select (form element).

How

Selecting a color

The color picker consists of two parts: an input (form element) and a popover.

When the user clicks on the input the popover is opened. To select a color, the user can click on a color tile in the popover. Once the user clicks, the popover is closed and the selected color is displayed in the input.

The user can browse through the colors by scrolling in the popover.

Closing the popover

The user can close the popover without selecting a color by clicking outside the popover.

Look and feel

The reset button is fixed on the bottom of the popover so that the user can easily reset the color without having to scroll down.

There is a clear visual distinction between different colors. Tones of the same color are visually grouped so that the user can easily identify tones from colors.

Highlights

When the user hovers over a color, the color is highlighted for extra feedback.

A selected color is marked as selected.

Default color & reset

By default, no color is selected.

Once a color is selected, the user can remove the color by resetting the color to the default.

Number of colors

The number and types of colors in the color picker depend on where the color picker is used; they are defined by design. The color picker can contain up to 50 colors.
